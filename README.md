# fake_forward_msg
用于发布虚假的群聊转发消息
<br>懂得都懂
<br>用法就是@bot造假@xxx{name}说xxx
<br>可以生成连续对话如@bot造假@xxx{name}说xxx@xxx{name}说xxx@xxx{name}说xxx
